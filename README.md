# 迷你旅行小程序

## 项目简介

这是一款基于微信原生小程序开发框架实现的一款旅游服务平台，旨在为用户提供一站式的旅游体验。用户可以通过小程序浏览热门景点、查看旅游攻略、预订门票、分享旅行见闻，还能参与论坛交流，获取个性化的旅游推荐。

### 核心功能

1. **首页推荐**：展示热门景点、美食推荐和精选旅游路线，根据用户位置和偏好提供个性化内容
2. **论坛社区**：用户可以发布旅行日记、分享攻略、提问交流，打造旅游爱好者的互动平台
3. **景点预订**：提供景点门票在线预订服务，支持多种支付方式，实时查看订单状态
4. **个人中心**：管理个人收藏、点赞记录，查看订单信息，编辑个人资料
5. **数据统计**：通过echarts可视化展示旅游数据，帮助用户了解热门目的地和出行趋势

### 技术栈

- **前端框架**：微信小程序原生开发
- **UI组件库**：Vant Weapp 1.11.3
- **数据可视化**：ECharts 5.4.3
- **工具库**：wxbarcode 生成条形码

### 项目特点

- 响应式设计，适配不同尺寸的移动设备
- 集成微信位置服务，提供精准的景点定位和导航
- 采用模块化开发，代码结构清晰，易于维护和扩展
- 完善的订单管理系统，支持订单查询、修改和取消
- 丰富的交互效果，提升用户体验
